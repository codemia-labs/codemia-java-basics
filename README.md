# Codemia Java Basics

> Apprenez les fondamentaux de Java grâce à des exemples simples, progressifs et directement exécutables.

## 📚 À propos

Ce dépôt constitue le premier module de la formation Java de **Codemia**.

L'objectif est d'apprendre les bases du langage Java à travers de courts exemples organisés par thème. Chaque chapitre est indépendant et peut être exécuté facilement.

## 🎯 Public visé

- Débutants en Java
- Étudiants
- Développeurs souhaitant revoir les fondamentaux

## 📖 Programme

- [ ] Introduction
- [ ] Variables
- [ ] Types primitifs
- [ ] Opérateurs
- [ ] Conditions
- [ ] Boucles
- [ ] Méthodes
- [ ] Tableaux

## 🚀 Prérequis

- Java 21 ou supérieur
- Un IDE Java (IntelliJ Community IDEA recommandé)

## 📂 Structure du projet

```
src/main/java/io/codemia/javabasics
```

Chaque chapitre possède son propre package ainsi qu'une classe `Main` permettant d'exécuter les exemples indépendamment.
