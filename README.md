# Codemia Java Basics

Premier module de la formation Java de **Codemia**, consacré à l'apprentissage des fondamentaux du langage Java.

Chaque chapitre est accompagné d'explications, d'exemples exécutables et d'un point d'entrée (`Main`) permettant de tester les différentes notions de manière indépendante.

---

# Objectifs

À la fin de ce module, vous serez capable de :

- comprendre les bases du langage Java ;
- déclarer et manipuler des variables ;
- utiliser les principaux types de données ;
- manipuler les opérateurs ;
- écrire des conditions ;
- utiliser les différentes boucles ;
- créer vos propres méthodes ;
- manipuler des tableaux.

---

# Public visé

Ce dépôt s'adresse principalement à :

- toute personne découvrant Java ;
- les étudiants ;
- les développeurs souhaitant revoir les fondamentaux du langage.

# Organisation du projet

```
src/main/java/io/codemia/javabasics
```

Chaque chapitre est organisé de la manière suivante :

- un `README.md` présentant la théorie ;
- une classe `Main` servant de point d'entrée ;
- plusieurs classes d'exemples illustrant chaque notion.

Cette organisation permet d'étudier chaque chapitre de manière indépendante.

---

# Prérequis

- Java 21 ou supérieur
- IntelliJ IDEA Community (recommandé)

---

# Exécution

Clonez le dépôt :

```bash
git clone https://github.com/codemia-labs/codemia-java-basics.git
```

Ouvrez ensuite le projet dans votre IDE Java préféré et exécutez la classe `Main` du chapitre souhaité.
