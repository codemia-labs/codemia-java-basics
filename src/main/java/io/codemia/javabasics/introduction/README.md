# Introduction

Bienvenue dans le premier chapitre de **Codemia Java Basics**.

Cette introduction a pour objectif de vous présenter le langage Java, son écosystème et l'organisation du projet avant de commencer les premiers exercices.

## Objectifs

À la fin de ce chapitre, vous serez capable de :

* Comprendre ce qu'est Java.
* Installer les outils nécessaires.
* Exécuter votre premier programme Java.
* Comprendre l'organisation de ce dépôt.

## Contenu

* Présentation de Java
* Installation du JDK
* Présentation d'IntelliJ IDEA
* Structure d'un projet Java
* Premier programme : `Hello World`

## Exécuter les exemples

Chaque chapitre possède sa propre classe `Main` afin de pouvoir exécuter les exemples de manière indépendante.
